# 🏓 Pong

A simple two-player Pong clone built with Python and pygame.

## Setup

```bash
pip install pygame
python Pong.py
```

## Controls

| | Up | Down |
|---|---|---|
| Player 1 | `W` | `S` |
| Player 2 | `↑` | `↓` |

## What I Learned

- Setting up a pygame window and game loop
- Handling keyboard input with `pygame.key.get_pressed()`
- Drawing shapes and text to the screen each frame
- Basic collision detection with `colliderect()`
- Managing game state with global variables (score, ball position, speed)
